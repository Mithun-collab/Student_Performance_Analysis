# Student Performance Dashboard

This project analyzes student academic performance using Power BI.

## Project Workflow

1. Data Cleaning using Python (Google Colab)
2. Data Transformation
3. Data Visualization using Power BI

## Tools Used

Python  
Pandas  
Power BI

## Dashboard Insights

• Study time improves performance  
• Higher education ambition correlates with higher grades  
• Absences negatively impact results
